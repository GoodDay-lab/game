WASD, стрелочки - управление

ESC - пауза
1. продолжить
2. настройки
3. выход из игры

x_scr, y_scr - крд на экране
x_road, y_road - крд на трассе

класс Авто:
1. x, y - свои координаты на ЭКРАНЕ
1. sprite - 3 текстуры (повороты влево, вправо, прямо)
2. speed - текущую скорость
3. max_speed - максимальную скорость
4. way - положение (лево прямо право), используется для указания текущего спрайта через индекс
5. time - время от начала разгона (обнуляется при отпускании газа)

класс Игрок:
1. id
2. x, y - свои координаты на ТРАССЕ

класс Объект:
1. x, y - свои координаты на ЭКРАНЕ
2. x, y - свои координаты на ТРАССЕ
3. sprite - текстура
